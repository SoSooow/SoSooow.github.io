块级元素、内联元素与内联块级元素的区别

2019-08-06
HTML

块级元素

特点:

  • 可设置宽高，宽度默认100%
  • margin、padding均有效
  • 自动换行，且独占一行（元素前后有换行符）

常见元素：div p h1~h6 ol ul dl table address blockquote form


内联元素

特点:

  • 设置宽高无效，宽度随元素内容变化
  • margin、padding左右有效，上下无效 (padding上下看上去有效，但对周围元素并无影响)
  • inline元素不会独占一行，多个相邻的行内元素会排列在同一行里，直到一行排列不下，才会新换一行

常见元素：a span br i em strong label q var cite code


内联块级元素

特点:

  • 能设置宽高
  • margin、padding均有效
  • 不会自动换行

常见元素：img input



可置换元素

一个内容 不受CSS视觉格式化模型控制，CSS渲染模型并不考虑对此内容的渲染，且元素本身一般拥有固有尺寸（宽度，高度，宽高比）的元素，被称之为置换元素。

  • 置换元素就是浏览器根据元素的标签和属性，来决定元素的具体显示内容。
  • 例如浏览器会根据img标签的src属性的值来读取图片信息并显示出来，而如果查看(X)HTML代码，则看不到图片的实际内容；又例如根据input标签的type属性来决定是显示输入框，还是单选按钮等。
  • HTML中的img、input、textarea、select、object都是置换元素。这些元素往往没有实际的内容，即是一个空元素。
